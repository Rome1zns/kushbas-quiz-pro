

# Көшбасшы Quiz — План реализации

## Обзор
Интерактивная quiz-платформа в стиле Kahoot для соревнований директоров школ на казахском языке. Оценка 5 управленческих компетенций с real-time синхронизацией через Supabase Realtime.

## Дизайн
- **Палитра**: тёмно-синий (#1a1f3d) + золотой (#f5a623) + белый
- **Стиль**: современный, энергичный, «соревновательный»
- **4 кнопки ответов**: красный треугольник, синий ромб, жёлтый круг, зелёный квадрат (как в Kahoot)
- **Шрифт**: Inter с поддержкой казахских символов (ә, ғ, қ, ң, ө, ұ, ү, h, і)

## База данных (Supabase / Lovable Cloud)
- **games** — id, pin (6-значный), status (lobby/active/finished), current_question, created_at
- **questions** — id, text_kk, indicator (enum 5 показателей), options (jsonb), correct_answer, order_num
- **players** — id, game_id, name, school, total_score, joined_at
- **answers** — id, player_id, question_id, selected_option, time_taken_ms, score_earned
- 15 предзагруженных вопросов на казахском (по 3 на каждый показатель)

## Экраны и функционал

### 1. Главная страница
- Логотип «Көшбасшы Quiz» со стилизованной короной
- Две кнопки: «Жаңа ойын құру» (хост) и «Ойынға қосылу» (игрок по PIN)

### 2. Хост: Лобби
- Генерация 6-значного PIN + QR-код
- Список подключившихся игроков (real-time через Supabase Realtime)
- Кнопка «Бастау» (Старт)

### 3. Игрок: Подключение
- Ввод PIN → имя → школа/регион
- Экран ожидания старта

### 4. Игровой процесс (15 вопросов)
- **Хост-экран**: вопрос + 4 варианта + 20-сек таймер + количество ответивших
- **Игрок-экран**: 4 большие цветные кнопки + таймер
- После ответа: «Жауабыңыз қабылданды» + анимация ожидания
- Баллы: до 1000 за вопрос (базовые × оставшееся_время / 20)
- После каждого вопроса: топ-5 лидерборд на 5 секунд
- **Звуки**: тикающий таймер, правильный/неправильный ответ, финальные фанфары (с кнопкой отключения)

### 5. Финальный экран — Хост
- ТОП-10 рейтинг с анимацией конфетти для победителя
- Горизонтальная гистограмма: средний балл группы по 5 показателям
- Самый сильный и слабый показатель группы

### 6. Финальный экран — Игрок (на телефоне)
- Место + общий балл
- Радар-диаграмма (Recharts) по 5 показателям
- Балл по каждому показателю (шкала 1–5)
- Автотекст: «Сіздің ең күшті көрсеткішіңіз: [X] және [Y]. Дамытуды қажет ететін бағыт: [Z].»

### 7. Админ-панель (простая)
- Просмотр и редактирование вопросов
- Добавление новых вопросов с привязкой к показателю

## Real-time синхронизация
- Supabase Realtime подписки на таблицы `players` и `games`
- Хост обновляет `current_question` в `games` → игроки получают новый вопрос
- Игроки записывают ответы в `answers` → хост видит счётчик ответов

## 5 показателей и 15 вопросов
Кейсовые ситуации на казахском по темам:
1. **Стратегиялық ойлау** — долгосрочное планирование, миссия школы, приоритизация
2. **Коммуникация** — общение с родителями, учителями, аким
3. **Командамен жұмыс** — делегирование, мотивация, конфликты
4. **Өзгерістерді басқару** — внедрение инноваций, сопротивление, адаптация
5. **Кері байланыс беру** — конструктивная критика, похвала, развитие учителей
